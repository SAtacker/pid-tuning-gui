# pid-tuning-gui
GUI app to tune and plot pid data from WALL-E robot

# Installation
`pip3 install -r requirements.txt`

# Usage
`python3 app.py`

# Dependenices
```
numpy==1.18.2
PyQt5==5.14.2
PyQt5-sip==12.7.2
pyqtgraph==0.10.0
```
